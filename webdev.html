Course Modules

Module 1: Introduction to Web Development

In this module, you will learn the basics of web development, including what the web is, how websites are structured, and more.

Module 2: HTML Basics

This module covers the basics of HTML, the standard markup language for creating web pages. You'll learn how to structure web pages with elements like headings, paragraphs, and images.

Module 3: CSS Styling

In this module, you'll learn how to style web pages using CSS. You'll cover styling for text, colors, layouts, and more!

Module 4: JavaScript Fundamentals

Learn the fundamentals of JavaScript, the programming language that adds interactivity to your websites. You'll cover variables, functions, loops, and events.

Module 5: Advanced JavaScript and Frameworks

This module dives into more advanced JavaScript concepts, as well as popular frameworks like React and Angular.
